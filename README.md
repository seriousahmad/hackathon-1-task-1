# Physical AI & Humanoid Robotics: The Future of Embodied Intelligence

This repository contains the textbook website for "Physical AI & Humanoid Robotics: The Future of Embodied Intelligence", built with Docusaurus 3.7.

## Features

- Premium book-like reading experience with professional typography
- Responsive design optimized for all device sizes
- Interactive components with smooth animations
- Dark/light mode with persistent settings
- Comprehensive textbook content with 3 main chapters
- Supplementary materials: Learning Outcomes, Weekly Breakdown, Assessments, and Hardware Requirements
- RAG chatbot placeholder for future AI integration

## Getting Started

### Installation

```bash
# Install dependencies
npm install
```

### Local Development

```bash
# Start development server
npm start

# Open http://localhost:3000 to view the textbook
```

### Build for Production

```bash
# Build static files
npm run build

# Serve built files locally
npm run serve
```

## Deployment

This site is configured for GitHub Pages deployment:

```bash
# Deploy to GitHub Pages
npm run deploy
```
